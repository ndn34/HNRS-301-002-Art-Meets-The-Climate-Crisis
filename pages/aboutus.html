About the Project

This site was created by Nicholas Nguyen and Kevin Fang for the course Art Meets the Climate Crisis. We wanted to build something interactive that could connect class ideas to everyday habits. Rather than focusing only on large abstract climate statistics, this project breaks the issue into smaller categories such as waste, electricity, transportation, and device production. Our goal was to make environmental impact feel more visible and personal.

Why We Chose a Website

We chose to present this project as a website because it allowed us to combine research, art, and interactivity in one place. A slideshow could summarize ideas, but a website gave us more room to create calculators, compare categories, and include artwork that reflects the environmental themes of each section. Since the Anthropocene is about systems and everyday behaviors, we thought an interactive format would help communicate those ideas more clearly than a static presentation.

Project Contributions

We initially divided the project by page while working together on the overall concept and design of the site. Nicholas created the Waste and Electricity sections and drafted the Conclusion and About Us pages. Kevin created the Home, Technology, and Transportation sections. As the project developed, we both expanded and refined different parts of the website to improve the layout, visuals, and clarity of the explanations.
